## How It Works (Simple)

```
1. Entry
   ├─ Place BUY LIMIT @ EP
   └─ Fill → Register + Place bracket orders (TP + SL)

2. Every Price Tick
   ├─ PROFIT MODE (price > EP)
   │  └─ QP = price - 0.25%
   │  └─ Trailing_SL = price - 1%
   │  └─ new_SL = max(existing, Trailing_SL, QP)
   │
   └─ LOSS MODE (price <= EP)
      └─ QP = disabled
      └─ Tighten SL based on drawdown
      └─ new_SL = max(existing, tightened)

3. Order Update
   ├─ If new_SL > old_SL
   │  └─ Cancel old → Place new SL order
   └─ Log change to MongoDB

4. Exit Detection
   ├─ price >= TP → TP_HIT
   ├─ price <= SL → SL_HIT
   ├─ price <= QP → QP_HIT
   └─ Gap detected → GAP_FORCED (market sell)

5. Audit Trail
   ├─ order_changes collection: All SL/TP updates
   └─ position_exits collection: Trade history + PnL
```
